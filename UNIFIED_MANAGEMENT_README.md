# نظام الإدارة الموحد - Unified Management System

## 🌟 الميزات الجديدة

### 📊 لوحة الإدارة الموحدة
- **واجهة تبويب متطورة**: 5 تبويبات رئيسية لإدارة شاملة
- **إحصائيات فورية**: معلومات محدثة في الوقت الفعلي
- **إجراءات سريعة**: اختصارات للمهام الشائعة
- **تحليل صحة النظام**: مؤشر شامل لحالة النظام

### 🔧 مكونات متقدمة
1. **AdvancedDataTable**: جدول بيانات محسن مع:
   - فرز وبحث متقدم
   - تحديد متعدد مع إجراءات جماعية
   - تصفية سريعة
   - إحصائيات مباشرة

2. **نماذج ذكية**: نماذج إدخال محسنة لـ:
   - المخازن والمواقع
   - المنتجات والفئات
   - طلبات التحويل
   - إدارة المخزون

3. **عرض التفاصيل**: نوافذ تفصيلية مع:
   - معلومات شاملة
   - تتبع زمني للأحداث
   - إجراءات حسب الصلاحيات

### 🛡️ نظام الصلاحيات
- **Admin**: وصول كامل لجميع الميزات

- **Warehouse Manager**: إدارة المخازن والمخزون
