## 🔧 الملفات المعدلة

### Schema والقاعدة
- `shared/schema.ts` - إضافة spare parts schemas
- `database-setup.sql` - جداول قطع الغيار
- `server/drizzle-storage.ts` - CRUD operations

### APIs
- `server/routes.ts` - endpoints جديدة لقطع الغيار
- WebSocket broadcasting للتحديثات الفورية

### الواجهات
- `client/src/pages/service-requests.tsx` - dropdown مجمع بالفئات
- `client/src/components/service-requests/` - مكونات قطع الغيار
- Real-time components مع React Query
