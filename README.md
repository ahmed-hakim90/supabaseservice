# نظام إدارة الخدمات المتقدم - SupabaseService

## 🎯 نظرة عامة

نظام إدارة شامل ومتطور للمؤسسات يوفر أدوات متكاملة لإدارة المستودعات، المخزون، النقل، المستخدمين والأمان. النظام مطور باستخدام أحدث التقنيات مع **إزالة كاملة للبيانات التجريبية** وربط جميع الوظائف بـ APIs حقيقية.

## 🚀 آخر التحديثات (سبتمبر 2025)

### ✅ تنظيف شامل للنظام
- **إزالة جميع البيانات التجريبية**: ربط 6 صفحات رئيسية بـ APIs حقيقية
- **5 APIs جديدة لمراقبة النظام**: مراقبة الأداء والصحة في الوقت الفعلي
- **وظائف تصدير حقيقية**: تصدير البيانات الفعلية بتنسيقات CSV و JSON
- **إشعارات ذكية ديناميكية**: تنبيهات تلقائية بناءً على حالة النظام
- **تحديث تلقائي**: تحديث البيانات كل 30 ثانية
- **حل جميع أخطاء TypeScript**: نظام خالٍ من الأخطاء البرمجية

## 🌟 الميزات الرئيسية

### 🏢 الإدارة الموحدة المحدثة
- **إدارة المستودعات**: تتبع وإدارة شامل للمستودعات والمواقع
- **إدارة المخزون**: تتبع المخزون في الوقت الفعلي مع تنبيهات المخزون المنخفض  
- **إدارة النقل**: تتبع عمليات نقل البضائع والأجزاء
- **إدارة المستخدمين**: نظام صلاحيات متقدم مع المصادقة الآمنة
- **إدارة العملاء**: قاعدة بيانات شاملة للعملاء والموردين

## 🚀 التشغيل السريع

### متطلبات النظام
- Node.js 18+ 
- npm أو yarn
- PostgreSQL (أو Supabase)

### تشغيل النظام
```bash
# استنساخ المشروع
git clone <repository-url>
cd SupabaseService

# تثبيت التبعيات
npm install

# إعداد متغيرات البيئة
cp .env.example .env
# قم بتحديث قيم قاعدة البيانات في .env

# تشغيل الخادم والعميل معاً
npm run dev
```

### الوصول للنظام
- **الواجهة الأمامية**: http://localhost:5173
- **خادم API**: http://localhost:3001  
- **إدارة النظام**: http://localhost:5173/dashboard/system-admin
